## Quick Screenshot Tips

**For Best Results:**
- Use full HD resolution (1920x1080)
- Clean desktop background
- Close unnecessary windows
- Maximize BuildSmartOS window
- Use Windows Snipping Tool (Windows + Shift + S)
- Crop to show just the app (no desktop clutter)

**Editing (Optional):**
- Add subtle drop shadow
- Blur sensitive data if needed
- Add arrows/callouts to highlight features
- Use tools like Canva or Photopea
